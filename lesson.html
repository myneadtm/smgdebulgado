Introduction to C++
⏱ ~20 minutes

C++ is one of the most powerful and widely used programming languages in the world. It was created by Bjarne Stroustrup in 1979 as an extension of the C language, adding object-oriented features while keeping C's performance and efficiency.

As a BS Civil Engineering student, you may wonder — why do I need to learn programming? The answer is simple: modern civil engineering relies heavily on software for structural analysis, cost estimation, material modeling, and automated calculations. Learning C++ gives you the foundation to write your own engineering tools.

How C++ Programs Work

1
Write Source Code
You type instructions in C++ using a text editor or IDE (like Code::Blocks or Dev-C++). This is your source file (.cpp extension).
2
Compile
A compiler translates your human-readable C++ code into machine language (binary) that the computer can execute. It also checks for syntax errors.
3
Link
The linker combines your compiled code with any libraries you used (like iostream) to create the final executable file.
4
Execute
Your program runs! The computer follows your instructions line by line, from top to bottom.

Variables & Data Types

          
⏱ ~25 minutes

        

      


      
A variable is like a labeled box in your computer's memory where you store a value. Before you can use a variable in C++, you must declare it — telling the compiler its name and the type of data it will hold.


      

        
📦

        
Civil Engineering Analogy: Think of a variable like a material storage bin on a construction site. You label each bin (name) and specify what it holds — cement bags (int), pipe lengths (double), room labels (string). You can't store cement in a pipe bin!

      


      
C++ Data Types

      

        
Data TypeStoresSizeExample

        
intWhole numbers (no decimal)4 bytesint bags = 50;

        
floatDecimal numbers (7 digits precision)4 bytesfloat gpa = 1.75f;

        
doubleDecimal numbers (15 digits precision)8 bytesdouble area = 125.50;

        
charSingle character1 bytechar grade = 'A';

        
stringText (multiple characters)Variablestring name = "Juan";

        
boolTrue (1) or False (0) only1 bytebool passed = true;

      


      

        
⚠️

        
Use double over float for engineering calculations. Float only has 7 decimal digits of precision — not enough for accurate engineering measurements. Double gives you 15 digits.

      


      
Declaring Variables — Syntax Rules

#include <iostream>

Input & Output (cin / cout)

          
⏱ ~25 minutes

        

      


      
So far our programs only display fixed values. Real programs need to interact with users — asking for input and giving output based on that input. In C++, we use cout for output and cin for input.


      

        

          
📤

          
cout — Output

          
Sends data TO the screen. Uses the << (insertion) operator. Think: "see out" = output.

        

        

          
📥

          
cin — Input

          
Reads data FROM the keyboard. Uses the >> (extraction) operator. Think: "see in" = input.

        

      


      

        
cement_cost.cpp — Civil Eng Example

        
#include <iostream>
using namespace std;

int main() {
    int    numBags;
    double pricePerBag, totalCost;

    // Ask user for input
    cout << "Enter number of cement bags: ";
    cin  >> numBags;

    cout << "Enter price per bag (PHP): ";
    cin  >> pricePerBag;

    // Calculate and display result
    totalCost = numBags * pricePerBag;
    cout << "Total Cost: PHP " << totalCost << endl;

    return 0;
}

      


      

        
🔴

        
Common Mistake: Using << with cin (it's >>) or >> with cout (it's <<). Always: cout << ... and cin >> ...

Operators & Expressions

          
⏱ ~25 minutes

        

      


      
Operators are symbols that perform operations on variables and values. In C++, there are four main types you must know for Week 1.


      

        
TypeOperatorsDescriptionExample

        
Arithmetic+ - * / %Math operationsarea = l * w;

        
Relational== != < > <= >=Compare two values → true/falsescore >= 75

        
Logical&& || !Combine conditionsx>0 && y>0

        
Assignment= += -= *= /=Store values in variablestotal += price;

      


      

        
🔢

        
Integer Division Warning: In C++, 7 / 2 gives 3, not 3.5 — because both values are int. The remainder is discarded. To get 3.5, at least one value must be a double: 7.0 / 2 = 3.5

      


      

        
area_calculator.cpp — Civil Eng Example

        
#include <iostream>
using namespace std;

int main() {
    double length, width, area;

    cout << "=== Room Area Calculator ===" << endl;
    cout << "Enter length (m): ";
    cin  >> length;
    cout << "Enter width  (m): ";
    cin  >> width;

    area = length * width;   // arithmetic operator *

    cout << "Floor Area: " << area << " sq. meters" << endl;

    // Relational operator: is area large enough?
    bool isLarge = area >= 20.0;
    cout << "Area >= 20 sqm: " << isLarge << endl;

    return 0;
}
