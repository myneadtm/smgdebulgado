Pseudocode Basics
Plan your logic before you code

Pseudocode is a plain-English description of what a program will do — written in a structured but informal way. It is not actual code and won't run on a computer. Its purpose is to plan your algorithm before writing any syntax.

💡
Why write pseudocode? It helps you focus on logic without worrying about syntax errors. Engineers use pseudocode to design algorithms on paper before implementing them in any programming language.

Pseudocode rules (no strict standard, but follow these guidelines):

KeywordMeaningExample
START / ENDBeginning and end of algorithmSTART GradeChecker
INPUT / READGet data from userREAD grade
PRINT / OUTPUTDisplay resultPRINT "Passed"
IF / ELSE / ENDIFDecision makingIF grade >= 75
FOR / ENDFORCounted repetitionFOR i = 1 TO 5
WHILE / ENDWHILEConditional repetitionWHILE count < 10
SET / LETAssign a valueSET total = 0

Here is a pseudocode example for a grade checker:

START GradeChecker READ grade IF grade >= 90 THEN PRINT "Excellent — A" ELSE IF grade >= 80 THEN PRINT "Good — B" ELSE IF grade >= 75 THEN PRINT "Passed — C" ELSE PRINT "Failed — study harder" ENDIF END GradeChecker

And here is the same logic converted to C++:

grade_checker.cpp
#include <iostream> using namespace std; int main() { int grade; cin >> grade; if (grade >= 90) cout << "Excellent — A" << endl; else if (grade >= 80) cout << "Good — B" << endl; else if (grade >= 75) cout << "Passed — C" << endl; else cout << "Failed" << endl; return 0; }
Excellent — A
📝
Civil Engineering Application: Before writing code to check if a beam's load exceeds its capacity, write pseudocode first: IF applied_load > allowable_load THEN PRINT "Unsafe — Redesign Required"

Flowcharts
Visualize your algorithm with diagrams

A flowchart is a graphical representation of an algorithm using standard shapes. It shows the flow of execution step by step — making it easy to understand and communicate your logic to others.

Standard Flowchart Symbols:

03
if / else if / else
Making decisions in your code

A program that always does the same thing isn't very useful. Conditional statements let your program choose what to do based on a condition — just like how we make decisions in real life.

💡
Think of it like a road sign: If the road is clear → go straight. Else if there's a detour → turn right. Else → stop.

The if statement checks a condition. If it's true, the code inside the braces runs. If it's false, it's skipped.

if (condition) { // runs if condition is TRUE } else if (another_condition) { // runs if first was false but this is TRUE } else { // runs if ALL above conditions were false }

Let's look at a real example — a grade checker:

grade_checker.cpp
#include <iostream> using namespace std; int main() { int grade; cout << "Enter your grade: "; cin >> grade; if (grade >= 90) { cout << "Excellent! You got an A." << endl; } else if (grade >= 80) { cout << "Good job! You got a B." << endl; } else if (grade >= 70) { cout << "You passed with a C." << endl; } else if (grade >= 60) { cout << "You passed with a D." << endl; } else { cout << "You failed. Please study harder." << endl; } return 0; }
Enter your grade: 85 Good job! You got a B.

Logical operators let you combine conditions:

⚠️
Common Mistake: Do NOT confuse = (assignment) with == (comparison). Writing if (x = 5) sets x to 5 — it does NOT check if x equals 5!
04
switch Statement
Clean multi-choice decisions

When you have many specific values to check against one variable, switch is cleaner than writing many else if blocks. Think of it like a menu — each case is one menu option.

switch (variable) { case value1: // code for value1 break; case value2: // code for value2 break; default: // runs if no case matched break; }
💡
Always include break! Without break, execution "falls through" to the next case even if it doesn't match.
day_checker.cpp
#include <iostream> using namespace std; int main() { int day; cout << "Enter day number (1-7): "; cin >> day; switch (day) { case 1: cout << "Monday" << endl; break; case 2: cout << "Tuesday" << endl; break; case 3: cout << "Wednesday" << endl; break; case 4: cout << "Thursday" << endl; break; case 5: cout << "Friday" << endl; break; case 6: cout << "Saturday" << endl; break; case 7: cout << "Sunday" << endl; break; default: cout << "Invalid day number!" << endl; break; } return 0; }
Enter day number (1-7): 5 Friday

switch works with int and char types — not with strings or floating-point numbers.

05
for Loop
Repeat a fixed number of times

A loop lets you repeat a block of code multiple times without writing it again and again. The for loop is used when you know exactly how many times to repeat.

for (initialization; condition; update) { // code to repeat }
PartWhat it doesExample
initializationRuns once at start — sets the counterint i = 0
conditionChecked before each repeat — loop continues while truei < 5
updateRuns after each repeat — changes the counteri++
for_loop.cpp
#include <iostream> using namespace std; int main() { // Print numbers 1 to 5 for (int i = 1; i <= 5; i++) { cout << "Count: " << i << endl; } // Calculate sum of 1 to 10 int sum = 0; for (int i = 1; i <= 10; i++) { sum += i; // same as: sum = sum + i } cout << "Sum 1-10: " << sum << endl; return 0; }
Count: 1 Count: 2 Count: 3 Count: 4 Count: 5 Sum 1-10: 55
🏗️
Civil Engineering use: A for loop can automate calculating beam loads for each floor: for (int floor = 1; floor <= 10; floor++) — calculates load for all 10 floors automatically.
06
while Loop
Repeat while a condition is true

The while loop repeats as long as a condition is true. Use it when you don't know in advance how many times to repeat — you repeat until something changes.

while (condition) { // code to repeat // must update something or loop runs forever! }
while_loop.cpp
#include <iostream> using namespace std; int main() { int number = 1; while (number <= 5) { cout << "Number: " << number << endl; number++; // IMPORTANT: update the variable! } // Countdown example int countdown = 5; while (countdown > 0) { cout << countdown << "... "; countdown--; } cout << "Launch!" << endl; return 0; }
Number: 1 Number: 2 Number: 3 Number: 4 Number: 5 5... 4... 3... 2... 1... Launch!
⚠️
Infinite Loop: If you forget to update the variable inside the loop, the condition never becomes false and the loop runs forever — freezing your program!
07
do-while Loop
Run at least once, then check

The do-while loop is like the while loop, but it always runs at least once — it checks the condition after the first execution. Perfect for menus or input validation where you need to run the code before checking.

do { // code runs at least once } while (condition); // ← note the semicolon!
do_while.cpp
#include <iostream> using namespace std; int main() { int choice; do { cout << "\n=== MAIN MENU ===" << endl; cout << "1. Calculate area" << endl; cout << "2. Calculate perimeter" << endl; cout << "0. Exit" << endl; cout << "Enter choice: "; cin >> choice; if (choice == 1) cout << "Calculating area..." << endl; if (choice == 2) cout << "Calculating perimeter..." << endl; } while (choice != 0); // repeat until user enters 0 cout << "Goodbye!" << endl; return 0; }
=== MAIN MENU === 1. Calculate area 2. Calculate perimeter 0. Exit Enter choice: 1 Calculating area... === MAIN MENU === ... Enter choice: 0 Goodbye!
LoopUse when...Checks condition
forYou know exact number of repeatsBefore each iteration
whileRepeat while something is trueBefore each iteration
do-whileMust run at least onceAfter each iteration
08
Nested Structures
Loops inside loops, conditions inside loops

Nesting means putting one structure inside another. You can put a loop inside an if, an if inside a loop, or even a loop inside another loop. This is very powerful but requires careful thinking.

nested.cpp
#include <iostream> using namespace std; int main() { // Multiplication table (nested for loops) for (int i = 1; i <= 3; i++) { for (int j = 1; j <= 3; j++) { cout << i * j << "\t"; // \t = tab space } cout << endl; // new line after each row } cout << endl; // Loop with if inside for (int i = 1; i <= 10; i++) { if (i % 2 == 0) { cout << i << " is even" << endl; } else { cout << i << " is odd" << endl; } } return 0; }
1 2 3 2 4 6 3 6 9 1 is odd 2 is even 3 is odd ... 10 is even
💡
break and continue: Use break to exit a loop early. Use continue to skip the current iteration and go to the next one.
09
Design → Code Integration
Putting it all together: pseudocode, flowchart, and C++ implementation

Now that you know all the tools — pseudocode, flowcharts, and C++ control structures — let us apply the full design process to a complete engineering problem.

🏗️
Problem: A civil engineering program should accept 5 concrete strength values from a sensor, compute their average, then classify it as "Safe", "Marginal", or "Unsafe" based on the threshold.

Step 1 — Write the Pseudocode:

START ConcreteCheck SET total = 0 FOR i = 1 TO 5 READ strength SET total = total + strength ENDFOR SET average = total / 5 IF average >= 30 THEN PRINT "Safe — Concrete meets standard" ELSE IF average >= 25 THEN PRINT "Marginal — Monitor closely" ELSE PRINT "Unsafe — Replace concrete" ENDIF END ConcreteCheck

Step 2 — Implement in C++:

concrete_check.cpp
#include <iostream> using namespace std; int main() { double total = 0, strength; // Step 1: Read 5 strength values for (int i = 1; i <= 5; i++) { cout << "Enter strength reading " << i << ": "; cin >> strength; total += strength; } // Step 2: Compute average double average = total / 5; cout << "Average strength: " << average << " MPa" << endl; // Step 3: Classify if (average >= 30) { cout << "Result: Safe — Concrete meets standard." << endl; } else if (average >= 25) { cout << "Result: Marginal — Monitor closely." << endl; } else { cout << "Result: Unsafe — Replace concrete." << endl; } return 0; }
Enter strength reading 1: 28 Enter strength reading 2: 31 Enter strength reading 3: 27 Enter strength reading 4: 30 Enter strength reading 5: 29 Average strength: 29 MPa Result: Marginal — Monitor closely.
